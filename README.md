# my_strace
Personal project my_strace
Implementation of a syscall tracer for linux ELF executables written in the C language.

## Compile the project:

```
git clone git@github.com:alexandremourere/my_strace.git
cd my_strace
make
```

## Usage:

```
./my_strace {command}
```
